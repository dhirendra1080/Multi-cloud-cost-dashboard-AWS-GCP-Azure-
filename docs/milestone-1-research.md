# Cloud Pricing Models

## 1. Pay-As-You-Go

- No upfront commitment
- Pay only for usage
- Best for unpredictable workloads

Supported By:

- AWS On-Demand
- Azure Pay-As-You-Go
- GCP On-Demand

---

## 2. Reserved Instances

- 1-year or 3-year commitment
- Up to 72% cost savings
- Best for stable workloads

Supported By:

- AWS Reserved Instances
- Azure Reserved VM
- GCP Committed Use Discounts

---

## 3. Spot / Preemptible Instances

- Up to 90% cheaper
- Can be interrupted
- Best for batch workloads

Supported By:

- AWS Spot Instances
- Azure Spot VMs
- GCP Spot VMs

# Cost Optimization Strategies

## Right Sizing

Reduce oversized VMs

## Reserved Pricing

Commit for long-term workloads

## Spot Instances

Use for fault tolerant workloads

## Auto Scaling

Scale down during low usage

## Storage Lifecycle

Move old data to cheaper tiers

- No single provider is cheapest
- GCP cheapest for compute
- Azure best for Microsoft workloads
- AWS largest service ecosystem
- Spot pricing reduces up to 90%
